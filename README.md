# JavaScript Basic Questions

## 1. What is the difference between var, let, and const?

`var` is function scoped and can be redeclared and reassigned. 
`let` and `const` are block scoped, but `let` can be reassigned while `const` cannot.

## 2. What is the spread operator (...)?

The spread operator (`...`) is used to expand elements of an array or properties of an object into another array or object.


## 3. What is the difference between map(), filter(), and forEach()?

`map()` creates a new array by transforming each element.
`filter()` creates a new array with elements that match a condition.
`forEach()` simply runs a function on each element without returning a new array.

## 4. What is an arrow function?

An arrow function is a shorter syntax for writing functions in JavaScript using `=>`.


## 5. What are template literals?

Template literals are a way to create strings in JavaScript using backticks (`` ` ``) instead of quotes. They allow variables and expressions to be inserted directly inside a string `${}`.
